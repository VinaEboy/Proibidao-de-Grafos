# Se não existe um PDF proibidão de Grafos, vamos criá-lo!

A ideia é fazer a resolução dos exercícios da lista de grafos no LaTeX. Cada arquivo .tex representa uma sessão de exercícios. 

As resoluções foram feitas por alunos contribuidores, ou seja, não são oficiais e podem conter erros. Se você identificar algum erro em algum exercício, submeta a correção para o repositório. Assim como, se você sabe a resolução de um exercício que ainda não foi feito, submeta ao repositório!
